# Traveling-Salesman-Problem-TSP-Branch-and-Bound-C-
Traveling Salesman Problem (TSP) – Branch and Bound (C++)

C++ implementation of the Traveling Salesman Problem (TSP) using the Branch and Bound algorithm.

The program finds the optimal route that visits each city exactly once and returns to the starting point by using cost matrix reduction and lower bound estimation to efficiently prune suboptimal solutions.

Key Features
-cost matrix reduction (rows and columns),
-lower bound calculation,
-branch pruning,
-optimal path reconstruction
